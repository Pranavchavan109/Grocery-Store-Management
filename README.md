# Grocery-Store-Management
Grocery Store Management System built using SQL/MySQL for handling products, customers, orders, and inventory with efficient database design and query optimization.


##  Project Overview

The Grocery Store Management System is a relational database project developed using MySQL to manage essential grocery store operations such as product inventory, customer records, sales transactions, and order management.

This project simulates a real-world retail environment and demonstrates practical SQL skills including database design, data normalization, relationships, and advanced querying techniques. The system ensures efficient data storage, integrity, and retrieval to support business decision-making.
